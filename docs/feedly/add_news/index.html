    Отбор новостей для сайта

    Сервис Feedly собирает новости АПК из нескольких десятков источников. В Feeldy можно просматривать и помечать новости, которые нужно вывесить на zzr.ru. Отмеченные новости попадают на обработку контент-менеджеру и через некоторое время появляются на zzr.ru.

    Для отбора новостей:

    1. Авторизуйтесь на сайте
    2. После авторизации вы увидите общую ленту новостей из всех источников
    3. Для показа новостей только из одного источника выберите его в списке слева
    4. Для просмотра анонса кликните на заголовок новости
    5. Пометить новость можно нажав на иконку слева от заголовка
    6. Посмотреть все помеченные для размещения на сайт новости можно на вкладке “Read Later”
