    Авторизация на feedly.com

    1. Запросите данные для входа у администратора сайта
    2. В браузере зайдите на https://feedly.com/
    3. В правом верхнем углу нажмите на кнопку Login.
    4. В открывшемся окне нажмите на зеленую кнопку Continue with Feedly.
    5. Введите в поле “email” и “password” полученные у администратора данные и нажмите Login.
