    Баннер

    Добавление баннера

    В настоящее время поддерживается только сквозное добавление баннера 100х100 пикселей в правую боковую колонку сайта. Для добавления баннеров нестандартных размеров отправьте заявку техническому специалисту.

    1. Авторизуйтесь на сайте.

    2. Выберите пункт Баннер в административном меню Материалы -> Добавить материал

    3. Заполните поля (обязательные поля помечены звездочкой в окне добавления баннера):

      • Название
        Название баннера используется только для служебных нужд и не будет видно пользователям сайта.

      • Изображение
        Собственно файл с баннером. Нажмите кнопку Выбрать файл, выберите файл с жесткого диска вашего компьютера, затем нажмите Загрузить.

        Форматы: png, gif, jpg, jpeg
        Размер изображения: от 100x100 до 500x500 пикселей
        Размер файла: до 150 Кб

      • Ссылка к баннеру
        Гиперссылка на сайт партнера.

      • Срок действия
        Здесь можно обозначить дату и время, когда баннер будет автоматически снят с публикации.

    4. После заполнения полей нажмите кнопку Сохранить внизу страницы.

    5. Зайдите на главную страницу сайта и убедитесь, что баннер появился на сайте.

    6. Кликните на баннер и убедитесь, что происходит переадресация на указанный сайт партнера.

    Редактирование баннера

    1. Авторизуйтесь на сайте.
    2. Нажмите на пункт административного меню Материалы
    3. В появившемся окне выберите Тип материала – Баннер и нажмите кнопку Фильтр
    4. Нажмите кнопку Редактировать напротив нужного баннера
    5. Исправьте необходимые поля.
    6. Нажмите кнопку Сохранить внизу страницы.

    Удаление баннера

    1. Авторизуйтесь на сайте.
    2. Нажмите на пункт административного меню Материалы
    3. В появившемся окне выберите Тип материала – Баннер и нажмите кнопку Фильтр
    4. Нажмите на стрелку рядом с кнопкй Редактировать напротив нужного баннера и выберите пункт Удалить

    Сортировка баннеров

    1. Авторизуйтесь на сайте
    2. Выберите пункт Баннеры в административном меню Материалы -> Сортировка
    3. С помощью “крестиков” задайте нужную последовательность баннерам
    4. Нажмите кнопку Сохранить порядок внизу страницы
